jQuery deferred

Deferred对象是Promise/A规范的一种实现，是jQuery1.5引入的，通过调用jQuery.Deferred()构造函数创建，提供了链式调用回调函数的功能，它能够注册回调函数到一个队列中, 依次调用该队列中的回调函数,并且能够将调链中上一个函数(同步或者异步)的返回状态传递到下一个回调函数中
