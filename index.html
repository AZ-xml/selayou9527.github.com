注意事项

  1. 这些字符属于unicode字符集，所以文档要使用utf-8
  2. 第一列用于HTML使用，需要格式&#8672;
  3. 第二列用于CSS，需要格式'\21E0'
  4. 第二列也可以用在JavaScript中，需要格式\u21E0

Sublime打开加入右键菜单

windows

Window比较简单，网上各种方法，基本原理就是往注册表里写入

Windows Registry Editor Version 5.00

[HKEY_CLASSES_ROOT\*\shell\SubLime]
@="edit with Sublime Text"

[HKEY_CLASSES_ROOT\*\shell\SubLime\Command]
@="C:\\Program Files\\Sublime Text 3\\sublime_text.exe \"%1\""

[HKEY_CLASSES_ROOT\Directory\shell\SubLime]
@="edit with Sublime Text"

[HKEY_CLASSES_ROOT\Directory\shell\SubLime\Command]
@="C:\\Program Files\\Sublime Text 3\\sublime_text.exe -a \"%1\""

把安装路径改成自己机器的，然后把文件保存为.reg，双击就可以了

你可能没注意的CSS单位

扶今追昔

CSS中的单位我们经常用到px、pt、em、百分比，px和pt不用多说

em

em是相对单位，参考物是父元素的font-size，具有继承的特点

如果字体大小是16px（浏览器的默认值），那么 1em = 16px

这样使用换算很复杂，尤其是和px对应，大家总结出了经验

body {
    font-size: 62.5%;

}

这样之后 1em = 10px 在布局等使用的时候好换算了很多
