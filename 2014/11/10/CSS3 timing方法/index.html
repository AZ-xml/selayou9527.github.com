Transition Timing Functions

之前博客CSS 实现打字效果中提到了CSS3 动画中的重要属性animation-timing-function，因为主要侧重打字效果的实现没有细说，这里详细聊聊

什么是timing function

timing function用来定义动画过程中属性值随着时间怎么变化，我们中学的时候画过很多曲线，尤其是在物理中接触过很多时间相关的曲线，我们可以方便的看到物体位移和时间的关系

同时我们也了解到，并不是所有的位移和时间都是线性关系，那是匀速运动，自由落体是加速运动

随着时间的增加单位时间内的位移增加，位移变化显著，说人话就是速度原来越快，单位时间内下降的距离原来越大

除了自由落体还有很多种时间－位移曲线，CSS可以帮我们模拟这些曲线

那么为什么要模仿这些曲线呢，所有的渐变都用线性渐变不行吗？举个简单的例子，我们做动画的时候要模拟自由落体不用CSS咋搞？

在CSS3中通过animation-timing-function可以指定属性随时间的变化方式

使用

我们平时在CSS中这样使用动画

@keyframe ani-name{
  0% {...},
  50% {...}
  100% {...}
}

.move{
  animation: ani-name 2s infinite;
}

我们对animation属性进行了合写，分开看有下面几个

  1. animation-name
  2. animation-duration
  3. animation-timing-function
  4. animation-delay
  5. animation-iteration-count
  6. animation-direction

其中animation-timing-function属性有下列值

  1. ease
  2. linear
  3. ease-in
  4. ease-out
  5. ease-in-out
  6. step-start
  7. step-end
  8. steps
  9. cubic-bezier

MDN 为每个画了相应的曲线，其实所有的效果都是不同参数的二阶贝塞尔曲线，感兴趣可以自己用神器cubic-bezier看看效果

  • ease cubic-bezier(0.25, 0.1, 0.25, 1.0)

  • liner cubic-bezier(0.0, 0.0, 1.0, 1.0)

  • ease-in cubic-bezier(0.42, 0.0, 1.0, 1.0)

  • ease-out cubic-bezier(0.0, 0.0, 0.58, 1.0)

  • ease-in-out cubic-bezier(0.42, 0.0, 0.58, 1.0)

  • step-start steps(1,start)

  • step-end steps(1,end)

  • cubic-bezier

从上面demo中可以清楚看到steps效果，其实就是分步，和俄罗斯方块往下掉一样，一帧一帧的动，而step-start相当于steps(1,start)，step-end相当于steps(1,end)

steps有两个参数

  1. 第一个肯定是分几步执行完
  2. 第二个油两个值
    • start 第一帧是第一步动画结束
    • end 第一帧是第一步动画开始
