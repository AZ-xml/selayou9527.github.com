简单回顾

回顾一下GO的几个要点

  1. 命令式的静态语言，需要编译执行，但编译速度十分快
  2. 自动处理垃圾回收
  3. 任何地方都是utf-8的，包括代码本身
  4. 所有Go文件以 package <packagename> 开始，独立执行的文件必须是 package main，入口函数一定是 main
  5. 语句结束不用;

关键字

Go语言有25个关键字

大部分看一眼就知道什么意思

  • func 定义函数
  • go 用于并行
  • select 用于选择不同类型的通讯
  • interface 定义接口
  • struct 定义抽象数据类型
  • chan 用于channel通讯
  • type 声明自定义类型
  • map 一种数据类型

其它也以后都会介绍到

定义变量

var关键字是Go最基本的定义变量方式，与其它多数语言不同的是，Go 变量类型在变量名之后

var name type

定义多个相同类型的变量，Go会为其赋值为该类型的默认值

var name1, name2, name3 type

定义变量并初始化

var name type = value

初始化多个相同类型的变量

var name1, name2,m name3 type = value1, value2, value3

也可以去掉类型，Go会根据值自动判断类型

var name1, name2,m name3 = value1, value2, value3

在函数内部还可以这样

name1, name2, name3 = value1, value2, value3

多个声明可以分组

var (
    name1    type1
    name2    type2
    name3    type3
)

一个特殊的变量名是 _，任何赋给它的值都会被丢弃

_, b := 34, 35

35 赋值给 b,同时丢弃 34

Go编译器对声明了却没有使用的变量会报错

package main 

func main() {
     var i int
}

常量

常量就是编译期间就能确定值的量，初始化后程序不能改变值

const name type = value

也可以忽略类型

const Pi = 3.14

可以定义数字、布尔、字符串类型的常量

数据类型

Boolean

布尔值的类型为bool，值为true或false，默认是false

数值类型

整型

数值类型分为有符号（uint）和无符号（int）两种。Go里面直接定义好位数类型：rune, int8, int16, int32, int64和byte, uint8, uint16, uint32, uint64。其中rune是int32的别称，byte是uint8的别称

赋值可以用八进制、十六进制或科学计数法:077,0xFF,1e3 或者 6.022e23 这些都 是合法的

浮点数

浮点数的类型有float32和float64两种，默认是float64

复数

Go支持复数，默认类型是complex128（64位实数+64位虚数）。还有complex64(32位实数+32位虚数)。

复数的形式为RE + IMi，其中RE是实数部分，IM是虚数部分，而最后的i是虚数单位

var c complex64 = 5+5i

类型的变量之间不允许互相赋值或操作，不然会在编译时引起编译器报错

字符串

Go中的字符都使用UTF-8编码，字符串通过双引号"定义，类型是string

var s= "xxxxxxx"

Go中字符串时不可变的，强行改变会编译报错

var s string = "hello"
s[0] = 'c'  ／／cannot assign to s[0]

使用+链接字符串，使用 ` 定义多行字符串

var s = "hello" + "K"

var b = `hello
        world`
