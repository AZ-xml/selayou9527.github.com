安装&启动

在Mac下安装命令为主的软件自然是使用神器homebrew

brew install mysql

安装完成后homewbrew会提示一些命令，如果想开机自启MySQL

To have launchd start mysql at login:

    ln -sfv /usr/local/opt/mysql/*.plist ~/Library/LaunchAgents

Then to load mysql now:

    launchctl load ~/Library/LaunchAgents/homebrew.mxcl.mysql.plist

或者也可以手工启动

mysql.server start

初始设置

MySQL安装完成后可以通过root账户登录了，这时候密码为空，很不科学，MySQL提供了一个初始安装工具

常用命令

DDL、DML语句和其他关系型数据库基本一样

服务

  1. mysql.server start 启动服务
  2. mysql.server stop 停止服务
  3. mysql.server restart 重启服务

登录

  1. mysql -u -p 登录MySQL
  2. exit; 退出
  3. select user(); 查看当前用户

查看信息类

  1. show databases; 查看服务器当前数据库
  2. use dbname; 选择数据库
  3. show tables; 查看当前数据库下的表
  4. discribe tabelname; 查看表的字段信息
  5. explain sql 分析sql语句性能

用户

新建用户

insert into mysql.user(Host,User,Password) values("localhost","username",password("userpassword"));
flush privileges;

删除用户

 delete from user where User="username" and Host="localhost"; 
 flush privileges;

修改用户密码

update mysql.user set password=password('newpassword') where User="username" and Host="localhost";

给用户授权

grant select,update on dbname.* to username@host identified by 'rootpassword'; 
grant all privileges on dbname.* to username@host identified by 'rootpassword'; 
flush privileges;

phpMyAdmin

使用命令有时候还是不够直观，有很多图形化的MySQL管理工具，介绍一下phpMyAdmin

phpMyAdmin时web版的MySQL管理工具，使用PHP写成，所以本机需要有PHP支持。项目首页有下载地址，下载之后解压，放到服务器指定的目录

在pgpMyAdmin下创建一个config文件夹，并且开放可写权限

mkdir config
chmod 777 config

然后访问项目的phpMyAdmin/setup/index.php文件，按照指示就可以完成了
