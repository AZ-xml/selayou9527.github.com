Mac nginx 80端口开机自启

文章目录
  1. 1. 安装
  2. 2. 开机自启
  3. 3. 解决办法
  4. 4. 附
    1. 4.1. nginx常用命令
    2. 4.2. homebrew常用命令

安装

再Mac下安装nginx很简单，使用神器homebrew

brew install nginx

这样nginx就会安装到目录

/usr/local/Cellar/nginx

配置文件在

/usr/local/etc/nginx/nginx.conf

开机自启

安装完成后homebrew会给出启动命令及开机启动的配置

mkdir -p ~/Library/LaunchAgents/  

cp /usr/local/Cellar/nginx/1.6.2/homebrew.mxcl.nginx.plist ~/Library/LaunchAgents/

launchctl load -w ~/Library/LaunchAgents/homebrew.mxcl.nginx.plist

正常情况下这样就可以，但是如果我们使用nginx监听80端口则不行，只能每次手工使用root账号或者sudo启动

sudo nginx

解决办法

其实解决办法很简单，给nginx管理员权限即可

sudo chown root:wheel /usr/local/Cellar/nginx/1.6.2/bin/nginx  
sudo chmod u+s /usr/local/Cellar/nginx/1.6.2/bin/nginx

附

nginx常用命令

  • nginx #启动nginx
  • nginx -s reload ＃重新加载nginx
  • nginx -s stop #关闭nginx
  • ps -ef|grep nginx #查找nginx进程

homebrew常用命令

  • brew install #安装
  • brew uninstall #卸载
  • brew serach #查找
  • brew list #列出使用homebrew安装的包
  • brew upgrade #升级使用homebrew安装的包
  • brew update #升级homebrew
