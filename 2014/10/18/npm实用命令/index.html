node升级到最新版本

node有一个模块叫n，是专门用来管理node.js的版本的，在控制台升级node到最新稳定版本只需要三条命令

 sudo npm cache clean -f
sudo npm install -g n
sudo n stable

安装模块

npm install <name> -g 把模块安装到全局

npm init 引导创建package.json文件，包括名称、版本、作者信息等

npm install <name> --save 安装模块的同时把信息写到package.json中

npm install <name> --save-dev 安装模块的同时把模块和版本号添加到devdependencies部分

管理模块

npm remove <name> 移除模块

npm update <name> 更新模块

npm ls 列出当前安装的了所有模块

npm root 查看当前模块的安装路径

npm root -g 查看全局模块的安装路径
