CSS 垂直居中

文章目录
  1. 1. line-height
  2. 2. display:table
  3. 3. 绝对定位＋负边距
  4. 4. 参考

css实现div内容垂直居中是个老话题了，在不同的场景下有不同的办法，简单介绍总结一下

line-height

这个几乎是单行文本最常用的方式了，把元素的height和line-height设置为一样，这样就实现的垂直居中

<style>
.t1{
    margin:10px;
    height:80px;
    line-height:80px;
    border:dashed 1px orange;
}
</style>
<div class="t1">单行文本垂直居中</div>
单行文本垂直居中

总结

  • 优点: 实在是太简单了
  • 缺点: 仅适用与单行文本，块元素无效，高度写到CSS中，不能改变

PS: 当然对于块元素如果可以的话，也可以设置为display:inline-block，这样可以用同样方式实现垂直居中

display:table

我们知道table布局时代完全没有垂直居中的困扰，valign="middle" 就搞定了，可以使用css把div显示的设置为table

<style>
.t2{
    margin:10px;
    display:table;
}

.t2-cell{
    display:table-cell;
    vertical-align:middle;
    height:80px;
    border:dashed 1px orange;
}

.t2-content{
    height:30px;
    width:60px;
    background:blue;
}
</style>


<div class="t2">
    <div class="t2-cell">
        <div class="t2-content"></div>
    </div>
</div>




总结

  • 优点:可以动态改变高度
  • 缺点
    1. 浏览器兼容性，只有现代浏览器和IE8+支持（也许在很多同学看来这不是问题了）
    2. 标签嵌套，需要额外的嵌套一层标签（比table要好一些。。。）
    3. 失去了div默认的占据一整行的块元素属性（莫非有人觉得正合我意）

绝对定位＋负边距

  1. 对外层元素设置position:relative 或 position:absolute
  2. 子元素设置position:absolute 和 top:50%
  3. 子元素设置 height:xx 和 margin-top:-xx/2

其实就是利用绝对定位先把子元素移动到垂直50%处，然后再上移自身高度的一半，造成居中的效果

<style>
.t3{
    margin:10px;
    position:relative;
    height:80px;
    border:dashed 1px orange;
}

.t3-content{
    position:absolute;
    top:50%;
    margin-top:-15px;
    height:30px;
    width:60px;
    background:blue;
}
</style>    

<div class="t3">
  <div class="t3-content"></div>
</div>


总结

  • 优点：浏览器兼容性好、结构简单

  • 缺点：元素高度必须确定，需要写到CSS中

PS：其实这里的子元素也可以设置为position:relative，原理都一样
