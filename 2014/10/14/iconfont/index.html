icon font

文章目录

所谓icon font是指将页面上使用的icon不使用图片，而是做成字体一样的东西。这是基于各个浏览器都对CSS3的 @font-face的支持，我们将icon也用font实现。

为什么使用iconfont

  1. 字体文件比图片小
  2. 矢量图形，在普通屏幕上适配，到高清屏幕后不会模糊
  3. 在更换theme等操作的时候可以简单通过CSS控制就可以改变icon的大小，颜色等
  4. 透明效果也兼容IE6

浏览器兼容性

目前各个浏览器都支持Can I Use的数据

可以看到IE及之前版本浏览器是部分支持，不同的浏览器支持的字体格式有所区别

  1. safari: 支持.ttf .otf
  2. Chrome: 支持.ttf .otf .woff
  3. Firefox: 支持.ttf .otf .woff
  4. Opera: 支持.ttf .otf .woff .svg
  5. IE8-: 支持 eot
  6. IE9+: 支持 .woff

可以看出大部分浏览器是支持woff(web open font format)的，这是w3c的推荐格式。

使用方式

前提是得有字体文件，首先声明@font-face

@font-face {
    font-family: 'emotion';
    src: url('emotion.eot'); /* IE9*/
    src: url('emotion.eot?#iefix') format('embedded-opentype'), /* IE6-IE8 */
         url('emotion.woff') format('woff'), /* chrome、firefox */
         url('emotion.ttf')  format('truetype'), /* chrome、firefox、opera、Safari, Android, iOS 4.2+*/
         url('emotion.svg#svgFontName') format('svg'); /*  iOS 4.1- */
}

对希望使用icon font的元素添加类

.icon{
    font-family: 'emotion';
    -webkit-font-smoothing: antialiased;
    -moz-osx-font-smoothing: grayscale;
}

后两句是为了防止出现字体图标锯齿现象。

然后在相应元素内写入对应的文字就会展示为图标了

<i class="icon">&#33</i>
