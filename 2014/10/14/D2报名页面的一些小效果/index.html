D2报名页面的一些小效果

最近看到了阿里巴巴D2前段论坛的报名页面，有几个有意思的小技巧，都是关于CSS3的，总结一下，万一哪天也用到了呢

图片hover变大

也就是照片墙部分，这个不了解css3的情况下我们会用个固定尺寸的父元素把图片保卫起来，然后把图片设置为绝对定位，hover的时候是用jQuery Animation等方式修改图片的尺寸和位置，使图片变大并且保持居中

有了CSS3后一切都简单了起来

<div class="c">
    <div class="a"></div>
    <div class="a"></div>
    <div class="a"></div>
    <div class="a"></div>
    <div class="a"></div>
    <div class="a"></div>
    <div class="a"></div>
    <div class="a"></div>
    <div class="a"></div>
</div>
<style>
.a{
  width:100px;
  height:100px;
  background:red;
  margin:10px;
  float:left;
}

.a:hover{
  background:blue;
  transform:scale(1.5);
  transition: all 0.5s;
}

.c{
  width:450px;
  overflow:hidden;
}
</style>
 











抛开兼容性问题不谈，使用CSS3的好处还是很多的

  1. 代码看起来精简了很多
  2. CSS3放大元素后还是占原来的位置，不用外层占位
  3. CSS3可以控制变化的原点（默认中心），我们不用显式控制元素位置
  4. CSS3动画看起来更流畅

元素翻滚而出

    

这个效果一看知道是用CSS3做的，但具体怎么做的很多人却说不出来，其实做法很简单，看代码

.a1{
  height:60px;
  background:#f0f000;
  margin:30px;
  transform-origin: 105% 0;
  transform: rotateY(90deg) rotateX(-90deg);
  transition: all 1.5s;
}

我们一开始把div旋转，修改transform-origin到右上角，点击按钮的时候添加一个class

.b1{
   transform: rotate(0);
}

把旋转置为0，利用transition属性，就可以看到我们的动画效果了

闪闪发光的logo



这个效果也是利用CSS3的transition做的，div在hover的时候添加一个遮罩层，看看遮罩层怎么做的

<div class="a2">
    <div class="b2"></div>
</div>
<style>
    .a2{
      position:relative;
      width:300px;
      height:60px;
     background:url(http://lsly1989.qiniudn.com/xxxasddbgfbTB194jGGpXXXXaTXXXXUg7ELXXX-300-60.png);
    }

    .a2:hover .b2{
      left:350px;
      transition:0.5s;
    }

    .b2{
      position:absolute;
      top:0;
      left:-350px;
      width:300px;
      height:60px;
      transform:skewx(-35deg);
      background:linear-gradient(0deg,rgba(255, 255, 255, 0),rgba(255, 255, 255, 0.8),rgba(255, 255, 255, 0));
    }
</style>

遮罩层使用了css3的线性渐变，hover的时候transition移动遮罩层搞定
